# CheapPcbPaddleKey
这是用PCB和标准螺丝、螺杆、磁铁拼凑成的简易双桨电键，成本很低（只需要不到6元）
![image](/mianpic.jpg)
PCB包含在立创eda工程文件中，可以免费打样。打样后pcb板按照下图红线剪开，桨的两小臂要用 ### 锉刀打磨后塞进轴承里
![image](/pcbpic.jpg)

成品前后视角
![image](/fwdpic.jpg)
![image](/backpic.jpg)

制作需要使用0.5mm的漆包线或者导线在如图所示地方焊接，两边都要焊接连接上。导线最好像弹簧那样弯曲一两圈后再焊接，避免导线对桨键回弹的影响
